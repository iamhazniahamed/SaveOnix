# SaveOnix - AI-Powered Smart Expense Tracking System

## Project Overview
SaveOnix is an AI-powered mobile application designed to automate expense tracking through SMS-based transaction extraction and intelligent categorization.

## Technology Stack
- **Frontend:** Flutter (Dart)
- **Backend:** Django REST Framework (Python)
- **Database:** PostgreSQL with Supabase
- **Version Control:** Git & GitHub

## Project Structure
- `/backend` - Django REST API
- `/frontend` - Flutter mobile app
- `/docs` - Documentation
- `/database` - Database scripts

## Setup Instructions
1. Clone repository
2. Setup backend: `cd backend && python manage.py runserver`
3. Setup frontend: `cd frontend/saveonix_app && flutter run`
